## Getting Started

### Prerequisites

Before contributing, ensure you have:

- Node.js >= 18.0.0
- npm >= 9.0.0
- Git
- MetaMask wallet
- Basic understanding of React, TypeScript, and Solidity
- Familiarity with Ethereum and Web3 concepts
